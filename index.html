# Quick Integration Guide

1

Get Your Client Credentials

Download the KeyLine App from Google Play Store, create a developer profile, and register your application to obtain your unique Client ID and allowed callback URIs.

2

Redirect to KeyLine Gateway

Redirect your web or mobile client authentication button trigger to the following production endpoint structure:

https://keyline-l1n2.onrender.com/oauth/authorize?client_id=YOUR_CLIENT_ID&redirect_uri=YOUR_REDIRECT_URI&response_type=code&scope=openid
3

Exchange Code for Access Token

Upon successful OTP multi-factor entry, KeyLine securely redirects back to your destination URL containing a temporary authorization transaction parameter token code. Exchange this directly inside your server layers securely.
